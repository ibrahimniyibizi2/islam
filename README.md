# Rwanda Islamic Hub

A comprehensive Islamic services platform for the Muslim community in Rwanda.

## Features

- **Nikah Services** - Islamic marriage ceremonies and registration
- **Funeral Services** - Islamic funeral arrangements and Janazah prayers
- **Islamic Education** - Quranic studies, Madrasa programs, and certification
- **Shahada Certificate** - Conversion to Islam and certificate issuance
- **Zakat & Charity** - Zakat calculation, collection, and distribution
- **Mosque Services** - Mosque registration, management, and community events
- **Islamic Halal Certification** - Halal food and business certification
- **Hajj & Umrah Services** - Pilgrimage registration and travel arrangements
- **Islamic Mediation** - Family and community dispute resolution

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- shadcn/ui

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

The application is configured for deployment with proper SPA routing support.
